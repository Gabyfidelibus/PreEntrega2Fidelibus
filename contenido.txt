Buenos días, paso a dejar mi Pre-Entrega en la cual incluyo el wireframe de las 5 páginas en su versión móvil y de escritorio, así como también los 5 archivos HTML (index, productos, comparador, nosotros y contacto) y el archivo CSS para darle estilo a todas las páginas del sitio web.

En los archivos HTML, utilicé etiquetas contenedoras como 'nav' para la barra de navegación superior, 'main' para el contenido principal de las páginas, 'footer' y 'header' para las respectivas secciones de las páginas, y 'div' para definir los contenedores genéricos. También utilicé etiquetas multimedia como 'img', enlaces con la etiqueta 'a' y etiquetas de texto como 'h1', 'h2', 'p' y 'span'. Utilicé las listas 'ul' para hacer barras de navegación, tablas con 'table' en la página de comparaciones y formularios con 'form' y sus respectivos 'label' e 'input' para la página de contacto.

En el CSS, hice uso de los selectores por clase, etiquetas, id y el selector universal (*) para el reset. Le di estilo a los textos con propiedades como 'font-family', 'font-size', 'color', 'font-weight', 'text-align', 'text-shadow' y 'font-style'. También le di estilo a imágenes, fondos, el posicionamiento de los diferentes elementos del sitio y el box-modeling con 'padding', 'margin', 'border', 'box-shadow', 'border-radius', 'width' y 'height', con tamaños tanto fijos (px y rem), como dinámicos (% , vh y vw).
